# Nintendo eShop UI Clone (Game Explorer App)

A responsive Nintendo eShop-inspired web application built with React and Tailwind CSS that displays video game data using the RAWG API.

## Features
- Browse and display video game data from the RAWG API  
- View game information including images and details  
- Responsive UI designed for mobile and desktop screens  
- Component-based layout for structured and reusable UI sections  

## Tech Stack
React, TypeScript, Tailwind CSS, RAWG API
